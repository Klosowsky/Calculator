# Calculator
Calculator application that supports JSON files.

This application reads json file and performs calculations (add,sub,mul,div,sqrt). Results are sorted in ascending order and saved into txt file.


Example:

Input:
{
"obj1": {
 "operator": "add",
 "value1": 2,
 "value2": 3
},
"obj2": {
 "operator": "sqrt",
 "value1": 16
}
} 

Output:
obj2: 4
obj1: 5
